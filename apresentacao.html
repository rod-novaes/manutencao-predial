O Desafio

Complexidade

Rastrear centenas de componentes e tarefas com múltiplas periodicidades (semanal, mensal, anual) de forma manual.

Planejamento Intensivo

Elaboração manual de planos de trabalho que consome tempo valioso da fiscalização técnica.

Mitigação de Riscos

Processos manuais aumentam o risco de desvios e esquecimentos, impactando a conformidade com o PMOC.

A Solução Proposta

Digitalizar o PMOC

Converter as regras, checklists e periodicidades do plano em um sistema inteligente, que serve como uma extensão digital do plano oficial.

Centralizar o Conhecimento

Criar uma base de dados única e segura para todos os planos, componentes e históricos de execução, consolidando a informação.

Automatizar o Planejamento

Delegar ao sistema a tarefa de calcular, agendar e organizar as manutenções futuras, gerando planos de trabalho precisos e automáticos.

Foco Estratégico: Esta é uma ferramenta de planejamento e controle para a COSEN. Ela não interfere nem substitui sistemas oficiais de OS, mas sim os alimenta com um escopo de trabalho validado.

Funcionalidades do Protótipo

Dashboard Visual

Painel inicial que agrupa e prioriza as tarefas vencidas, do mês atual e do próximo, com um calendário interativo.

Estrutura Fiel ao PMOC

Organização em Grandes Áreas, Sistemas e Componentes, refletindo a lógica do plano oficial.

Agendamento por Tarefa

Cada tarefa possui sua própria periodicidade, garantindo conformidade com as especificações técnicas do PMOC.

Flexibilidade de Cadastro

Permite adicionar tarefas específicas para um componente individual, além das tarefas herdadas do sistema padrão.

Geração Automática

O sistema calcula os próximos vencimentos com base no histórico, mantendo o cronograma sempre atualizado.

Relatórios de Trabalho

Gera um plano de trabalho formatado para qualquer período, pronto para ser entregue às equipes terceirizadas.

Fluxo de Trabalho Proposto

1. Planejar

A COSEN alimenta o sistema com os planos do PMOC e o inventário de componentes.

2. Gerar

O gestor gera o relatório de tarefas pendentes para o período desejado.

3. Executar

O relatório é entregue à contratada com o escopo dos serviços a serem executados.

4. Registrar

O fiscal registra a conclusão, e o sistema agenda o próximo ciclo automaticamente.

Vantagens da Adoção

  • Mitigação de Riscos: Reduz significativamente o risco de esquecimento de tarefas críticas, garantindo maior conformidade com o PMOC.
  • Otimização do Tempo: Automatiza o processo de planejamento, liberando a equipe técnica para focar na fiscalização, análise e melhoria contínua.
  • Padronização: Garante que o escopo de trabalho comunicado às contratadas seja sempre completo, preciso e uniforme.
  • Histórico Confiável: Constrói uma base de dados estruturada para auditorias e tomadas de decisão futuras.

Limitações Atuais e Oportunidades Futuras

Limitações

  • Arquitetura Standalone: O aplicativo foi projetado para operar sem depender da infraestrutura de servidores e bancos de dados do TSE. Os dados do protótipo, por enquanto, são salvos localmente no navegador.
  • Aplicação Monousuário: Pelo mesmo motivo, o protótipo não é multiusuário e não possui sistema de autenticação, sendo destinado ao uso em uma estação de trabalho por vez.
  • Fluxo Manual: A integração com sistemas externos (ex: IBM Máximo) não está prevista. Ela pode ser feita via registro manual pelo fiscal.
  • Foco em Preventivas: O escopo atual cobre exclusivamente as manutenções preventivas.
  • Aplicativo Desktop: O protótipo atual é uma versão apenas para desktop.

Oportunidades

  • Gestão de Dados Local: Caso a continuidade do desenvolvimento seja aprovada, serão implementados botões para Salvar/Abrir os dados em um arquivo local (JSON).
  • Não Dependência: A funcionalidade de gerar backups em planilha (XLSX) deve ser implementada. A ideia é não gerar dependência exclusiva do aplicativo para guardar o histórico de manutenções. O aplicativo deve ser uma ferramenta de auxílio, não uma prisão.
  • Evolução para Dashboards: Criação de painéis com indicadores de desempenho (KPIs) de conformidade e execução.
